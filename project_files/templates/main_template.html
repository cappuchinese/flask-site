Fasta reader

This theme is focussed on creating our own website using the web application framework Flask. Flask was formerly used as a simple wrapper for Werkzeug and Jinja, but has become one of the most popular Python web frameworks.

For Bio-informatics, it is common to use files filled with information about the molecular biology in organisms. To read such files is very complicated to do by hand. To make it easier to read, we can build a website that reads a file and gives the user the wanted data. This website is build to specifically read FASTA formatted files.
