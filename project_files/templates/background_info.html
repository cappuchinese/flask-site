Information on FASTA files

A FASTA formatted file contain two things: a header and a sequence.
The usual file format is 70 characters long, except for the header; headers are identified by starting the line with a ">" (greater-than) symbol and contain information regarding the sequence (which animal it's from, which gene, etc.).
Some files can contain multiple sequences and are separated by the headers.
The sequence is either a nucleotide sequence or a amino acid sequence. In the example shown below, a nucleotide sequence is used.
